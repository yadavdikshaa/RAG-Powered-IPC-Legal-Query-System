# RAG-Powered-IPC-Legal-Query-System
A Retrieval-Augmented Generation (RAG) system that helps lawyers and individuals find relevant Indian Penal Code (IPC) sections based on legal queries. It uses FAISS for vector search, Hugging Face MiniLM-L6-v2 for embeddings, and Groq LLaMA-3 for legal reasoning via FastAPI.
